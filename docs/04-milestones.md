# Kissaten Tycoon — Milestones

## Milestone 1 — Balanced Simulation Model

**Outcome:** The economy is mathematically possible and believable.

**Required work:**

- finalize units
- calibrate default values
- test break-even behavior
- define formula bounds
- define bankruptcy behavior

**Demo:** Run one manually verified quarter and explain every calculation.

---

## Milestone 2 — Tested Engine Prototype

**Outcome:** The simulation works from the terminal without AI or UI.

**Required work:**

- game state
- decisions
- quarter resolution
- state transition
- seeded randomness
- reconciliation tests
- extreme-decision tests

**Demo:** Play four quarters from a CLI using hardcoded competitor and event values.

---

## Milestone 3 — Playable Browser Prototype

**Outcome:** A user can complete the deterministic game in a browser.

**Required work:**

- dashboard
- decision controls
- current state
- quarter results
- charts
- mocked agents

**Demo:** Start a game, resolve four quarters, and reach the final result.

---

## Milestone 4 — AI Professor Integration

**Outcome:** GPT-5.6 explains decisions and consequences.

**Required work:**

- Professor schema
- trusted engine context
- output validation
- fallback explanation
- concept and Socratic question

**Demo:** Show the Professor explaining a capacity constraint.

---

## Milestone 5 — Adaptive Competitor and Market

**Outcome:** The world reacts to the player.

**Required work:**

- Competitor Agent

- timeouts and fallbacks

**Demo:** Lower the price and show the competitor responding defensively.
